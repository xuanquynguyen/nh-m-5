Các bạn có thể sử dụng copy 2 file .mdf và .ldf vào đường dẫn sau:
C:\Program Files (x86)\Microsoft SQL Server\MSSQL10_50.SQLEXPRESS\MSSQL\DATA
nếu là bản SQL 32bit
C:\Program Files\Microsoft SQL Server\MSSQL10_50.SQLEXPRESS\MSSQL\DATA
nếu là bản SQL 64bit

sau đó trong SQL nhấn chuột phải vào Databases chọn Attack rồi add file mdf và nhấn OK

hoặc dùng Script, nếu là bản SQL 2008 R2 thì chọn bản R2 mở lên sau đó chỉ việc Execute
